## 🗑️ 删除文件接口：`deletefilekb`

用于用户删除指定文件。

### 请求方式：

`POST`

### 接口地址：

`/deletefilekb`

### 鉴权说明：

需携带用户身份验证 Token

### 请求参数：

| 参数名 | 类型   | 是否必填                | 说明                                |
| ------ | ------ | ----------------------- | ----------------------------------- |
| docId  | string | 是                      | 要删除的文档 ID                     |
| token  | string | 是（从 AuthGuard 获取） | 用户身份标识（从登录 Token 中解析） |

### 请求示例：

```json
{
  "docId": "abc123456"
}
```
