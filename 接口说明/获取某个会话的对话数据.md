## 🗑️ 获取某个会话的对话数据：`singlechatdata`

用于用户获取某个会话的对话数据。

### 请求方式：

`GET`

### 接口地址：

`/singlechatdata`

### 鉴权说明：

需携带用户身份验证 Token

### 请求参数：

| 参数名    | 类型   | 是否必填                | 说明                                |
| --------- | ------ | ----------------------- | ----------------------------------- |
| token     | string | 是（从 AuthGuard 获取） | 用户身份标识（从登录 Token 中解析） |
| sessionId | string | 是                      | 会话id                              |
